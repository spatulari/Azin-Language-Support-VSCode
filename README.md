# Azin Language Support

Visual Studio Code extension providing syntax highlighting and basic language
configuration for the [**Azin**](https://github.com/Alex12-Git200/Azin) programming language.

---

## Features

- Syntax highlighting for `.az` files
- Keyword highlighting:
  - `return`
  - `if`
  - `while`
  - `for`
- Support for:
  - String literals
  - Numeric literals
  - Function calls using `function@module(...)`
  - Module identifiers
  - Parentheses and basic operators
- Automatic language detection for Azin source files

---

## Example

```az
out@io("Hello World");
return 5;
```
